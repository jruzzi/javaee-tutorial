23.11 Using a Managed Bean in a Facelets Page

To use the managed bean in a Facelets page:

  1. Create a form that uses user interface elements to call its methods and display their results.

    This example provides a button that asks the user to type a name, retrieves the salutation, and then displays the text in a paragraph below the button:

    <h:form id="greetme">
   <p><h:outputLabel value="Enter your name: " for="name"/>
      <h:inputText id="name" value="#{printer.name}"/></p>
   <p><h:commandButton value="Say Hello"
                       action="#{printer.createSalutation}"/></p>
   <p><h:outputText value="#{printer.salutation}"/></p>
</h:form>
